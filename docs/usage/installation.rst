Installation
============

From Source
^^^^^^^^^^^

First, clone this repository with

.. code-block:: bash

    $ git clone https://github.com/chazeon/qha-cij.git

Then, install with ``pip`` with

.. code-block:: bash

    $ pip install .

or to install in development mode use

.. code-block:: bash

    $ pip install -e .

Dependencies
^^^^^^^^^^^^

1. QHA
